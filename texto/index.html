O que é uma tag HTML?

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

O que aprendi.

Aprendi que o HTML Linguagem de Marcação de Texto, contém uma marca de início e outra de fim para que o navegador possa renderizar uma página. Apesar de receber o título de linguagem na verdade não se trata de uma liguagem de programação e sim um paradigma declarativo no qual são usadas tags para definir marcações especificas no texto. Atravéz das tags podemos definir um título e sua hierarquia no texto é possível também definir a forma com a qual queremos que o usuário vizualize o texto, como por exemplo se o texto aparecerá em negrito, formato itálico, sublinhado, , isso e muito mais é possível utilizando as tags HTML .

Atributos.

Nas tags existem os atributos que são palavras especiais dentro das tags que controlam o comportamento do elemento.
Os atributos podem ser:

  • Globais:
    São aceitos por todas as tags,temos como exemplo as palavras: class, id, lang, style e algumas outras;
  • Especificos:
    Que somente algumas tags possuem, como exemplos temos: src, disabled, href, label, etc…
